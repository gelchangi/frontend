# Screenshots

Add screenshots of the DecaDrive frontend here for your demo presentation.

Recommended screenshots:

1. **Home Page** - Landing page with logo and CTA
2. **Lessons Page (Light Mode)** - Lesson cards with search/sort
3. **Lessons Page (Dark Mode)** - Same page in dark theme
4. **Cart Page** - Cart with items and summary
5. **Checkout Page** - Form with validation
6. **Order Success Page** - Confirmation screen

## How to Take Screenshots

1. Run the app locally
2. Navigate through each page
3. Use browser screenshot tools or:
   - **macOS**: `Cmd + Shift + 4` (select area)
   - **Windows**: `Win + Shift + S`
4. Save images to this directory

## Naming Convention

Use descriptive names:

- `01-home-page.png`
- `02-lessons-light-mode.png`
